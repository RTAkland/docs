# CarryEntity

> 这个插件是可以将一个生物搬起来，安装Shift + 右键一个生物即可将其搬起来，搬起来后按下Shift即可将其放下

# 展示

![CarryEntity](../images/carryentity/carryentity-showse.gif)

# 使用

使用`/carryentity enable/disable` 可以调整自己的开关
