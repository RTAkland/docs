# 自定义生物目标

可以设置生物的目标, 设置之后就可以让生物攻击目标生物

效果如下

    <source src="https://static.rtast.cn/static/mobcustomtarget-showcase.mp4" type="video/mp4">
</video>

# 使用

使用命令/set-target <攻击者(可以设置多个, 可以使用uuid或者@选择器)> <目标生物(只能设置一个)>

命令的权限节点是: `mobcustomtarget.execute`
